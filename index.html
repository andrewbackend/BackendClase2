Introducción a Git y GitHub

¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite a los desarrolladores llevar un seguimiento de los cambios en su código fuente durante el desarrollo de software.

¿Qué es GitHub?

GitHub es una plataforma basada en la web que utiliza Git como su sistema de control de versiones. Proporciona herramientas adicionales como colaboración, gestión de proyectos y almacenamiento de código en la nube.
